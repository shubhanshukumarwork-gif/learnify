# 🚀 Learnify – Full-Stack E-Learning Platform

Learnify is a modern full-stack e-learning platform designed to make learning interactive, engaging, and community-driven. It includes quizzes, real-time scoring, leaderboards, and a discussion forum to enhance student collaboration and motivation.

## 📌 Features

- 🎯 Interactive quiz system with real-time score tracking  
- 🏆 Leaderboard to encourage competition among learners  
- 💬 Discussion forum with full CRUD functionality  
- 👥 Community-driven learning experience  
- ⚡ Fast and responsive UI for all devices  
- 🔐 Secure backend with Express.js and MongoDB

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/your-username/learnify.git
cd learnify

2. Install dependencies
Frontend
cd client
npm install
Backend
cd server
npm install
3. Setup environment variables

Create a .env file in the server folder:

MONGO_URI=your_mongodb_connection_string
PORT=5000
JWT_SECRET=your_secret_key
4. Run the project
Start backend
cd server
npm start
Start frontend
cd client
npm run dev
